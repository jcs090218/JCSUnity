JCS_2DAnimator

Hold sequence of animations and play it by condition.

Variables

NameDescription
mAnimationTimeProductionHow fast the animation plays.
mAnimationsAll the animations this animator holds.
m2DAnimDisplayHolderHold animation displayed frame event.

Functions

NameDescription
DoAnimationPlay the animation base on the animation ID.
PlayOneShotPlay one animation once and go back to original animation.
IsInStateCheck what animation is currently playing by this animator.
PlayAnimationInFramePlay the animation with current frame.
StopAnimationInFrameStop animation with current frame.
