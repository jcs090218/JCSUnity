JCS_ScaleEffect

Scale the object to a specific value.

Variables

NameDescription
mScaleXDo the scale on x axis.
mScaleYDo the scale on y axis.
mScaleZDo the scale on z axis.
mScaleValueHow much it scales on each axis.
mScaleFrictionHow fast it scales on each axis.
mAutoAddEventAdd event to event trigger system.
mActiveEventTriggerTypeEvent trigger type to active the the slide effect.
mDeactiveEventTriggerTypeEvent trigger type to deactive the the slide effect.

Functions

NameDescription
JCS_OnMouseOverCall it when is on mouse over.
JCS_OnMouseExitCall it When is on mouse exit.
ActiveActive the effect.
