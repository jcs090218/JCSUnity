JCS_FreezeTransformAction

Action that freeze the game object position, rotation and scale.

Variables

NameDescription
mActiveIs this action active?
mPositionToFreezePosition where this game object freeze at.
mRotationToFreezeRotation where this game object freeze at.
mScaleToFreezeScale where this game object freeze at.
mIsLocalPositionFreeze in the local space.
mIsLocalRotationFreeze in the local space.
mFreezePositionFreeze the position in each axis.
mFreezeRotationFreeze the rotation in each axis.
mFreezeScaleFreeze the scale in each axis.
