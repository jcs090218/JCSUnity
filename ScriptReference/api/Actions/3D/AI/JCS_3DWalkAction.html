JCS_3DWalkAction

Simulate the walk action in 3D space.

Variables

NameDescription

AllowOverlapDestinationAllow each walk action having the same destination.
OverlapDistanceDistance that would count as overlap destination.
AcceptRemainDistanceWhat value count as path complete action.
MinOffDistanceMinimum randomly add vector with magnitude of distance at target position.
MaxOffDistanceMaximum randomly add vector with magnitude of distance at target position.
SelfDistanceSelf distance without target transform interact.
RangeDistanceRange that enemy will try to get close to.
AdjustRangeDistanceRandomly adjusts the range distance.
