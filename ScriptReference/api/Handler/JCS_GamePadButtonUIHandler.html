JCS_GamePadButtonUIHandler

Variables

NameDescription

mKDisabledSpriteSprite when button is disabled. (Keyboard)
mJNormalSpriteSprite when button is normal. (Joystick)
mJHighlightedSpriteSprite when button is hover. (Joystick)
mJPressedSpriteSprite when button is pressed. (Joystick)
