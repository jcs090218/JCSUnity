JCS_Button

Button wrapper for more usable functionalities.

Variables

NameDescription
mButtonTextText component under the button.
mButtonSelectionButton Selection for if the button that are in the group.
mAutoListenerAuto add listner to button click event?"
mDialogueIndexIndex/ID for record the dialogue instance.
mInteractableIs the button interactable or not.
mInteractColorColor tint when button is interactable.
mNotInteractColorColor tint when button is not interactable.

Example

ExampleButton.cs

  class ExampleButton : JCS_Button {
      public override void JCS_OnClickCallback() {
          Debug.Log("On click!"");
      }
  }
