JCS_TextTimer

Render timer in the text.

Variables

NameDescription
mActiveTimer active or not active.
mCurrentHoursCurrent hours in the game.
mCurrentMinutesCurrent minutes in the game.
mCurrentSecondsCurrent seconds in the game.
mDelimiterTextText add between each unit.
mRoundUpDo round up instead of round down.
mHideWhenZeroHide the number when is zero.
mTimeTypeType of the delta time.
mHourSoundSound played when hours get reduced.
mMinuteSoundSound played when minutes get reduced.
mSecondSoundSound played when seconds get reduced.
