JCS_SequenceSlidePanel

Sequently toggle the panels action.

Variables

NameDescription
mToggleOrExitButtonOptional choice, instead of using the auto detection. (JCS_EmptyButton are the recommaned default class to use.)
mSlideButtonsSequence of buttons with slide effect component with in the tranform.
mAreaEffectsArea that also control with this, please do it manully. (only in children)
mSpaceTimeTime to active one button animation.
mDeltaTimeTypeType of the delta time.
mExitByNoOverGUIExit when mouse is not on this panel.
mActiveClipAudio when active the animation the outer panel.
mDeactiveClipAudio when exit the animation the outer panel.
mInsideActiveClipAudio when active the animation inside the panel.
mInsideDeactiveClipAudio when exit the animation inside the panel.
