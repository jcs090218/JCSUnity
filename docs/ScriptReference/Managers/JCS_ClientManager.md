# JCS_ClientManager

Manager manage the clients that connected to this client.

## Variables

| Name     | Description                             |
|:---------|:----------------------------------------|
| mClients | Clients that interact with this client. |

## Functions

| Name         | Description                         |
|:-------------|:------------------------------------|
| AddClient    | Add the client to the manager.      |
| RemoveClient | Remove the client from the manager. |
