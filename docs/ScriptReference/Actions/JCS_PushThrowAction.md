# JCS_PushThrowAction

Action that throws the object to a position.

## Variables

| Name           | Description                |
|:---------------|:---------------------------|
| mEffect        | Do effect?                 |
| mAngle         | Angle you want to throw to |
| mG             | Speed of this projectile.  |
| mDeltaTimeType | Type of the delta time.    |

## Functions

| Name     | Description                                 |
|:---------|:--------------------------------------------|
| SetStart | Call this function to start the projectile. |
