# JCS_HopEffect

Make object hop for a force.

## Variables

| Name     | Description                                         |
|:---------|:----------------------------------------------------|
| mForce   | How much force to jump away from current position.  |
| mGravity | How fast the object goes back to original position. |
| mAxis    | Hop axis.                                           |

## Functions

| Name     | Description                |
|:---------|:---------------------------|
| StartHop | Start the hopping process. |
