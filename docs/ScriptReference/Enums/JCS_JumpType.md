# JCS_JumpType

List of type of jumping action.

## Values

| Name              | Description       |
|:------------------|:------------------|
| BASIC_JUMP        | Normal jump       |
| DOUBLE_JUMP       | Double jump       |
| TRIPLE_JUMP       | Triple jump       |
| DOUBLE_JUMP_FORCE | Force double jump |
| TRIPLE_JUMP_FORCE | Force triple jump |
