# JCS_PathActionType

List of path action type.

## Values

| Name       | Description                                    |
|:-----------|:-----------------------------------------------|
| INCREMENT  | Increment the path points.                     |
| DECREMENT  | Decrement the path points.                     |
| INC_OR_DEC | Random the increment or decrement path points. |
| RANDOM_ALL | Random all the path points.                    |
