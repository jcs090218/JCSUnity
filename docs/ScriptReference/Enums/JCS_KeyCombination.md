# JCS_KeyCombination

Key combination.

## Values

| Name           | Description                     |
|:---------------|:--------------------------------|
| NONE           | Not specify                     |
| ALT            | Alternate key                   |
| CTRL           | Control key                     |
| SHIFT          | Shift key                       |
| ALT_CTRL       | Alternate + Control key         |
| ALT_SHIFT      | Alternate + Shift key           |
| CTRL_SHIFT     | Control + Shift key             |
| ALT_CTRL_SHIFT | Alternate + Control + Shift key |
