# JCS_RecvPacketType

List of packet handle for receiving.

## Values

| Name            | Description                  |
|:----------------|:-----------------------------|
| PING            | Ping packet                  |
| HELLO           | Hello packet (testing)       |
| LOGIN_PASSWORD  | Packet for login             |
| PLAYER_LOGGEDIN | Packet for confirm login     |
| MOVE_PLAYER     | Packet for moving the player |
