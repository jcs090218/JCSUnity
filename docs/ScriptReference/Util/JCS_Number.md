# JCS_Number

Number extensions

## Functions

| Name   | Description                                                                         |
|:-------|:------------------------------------------------------------------------------------|
| GetP   | Return the percentage.                                                              |
| Delta  | Delta the `num` with `val` and clamp the result with `min` and `max`.               |
| DeltaP | Delta the `num` with `val` by percentage and clamp the result with `min` and `max`. |
