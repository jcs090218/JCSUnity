# JCS_2DAnimDisplayHolder

Hold an animation state for few seconds.

## Variables

| Name      | Description                      |
|:----------|:---------------------------------|
| mHoldTime | How long to hold this animation. |

## Functions

| Name          | Description                        |
|:--------------|:-----------------------------------|
| HoldAnimation | Hold an animation for few seconds. |
| StopHolding   | Stop holding the animation.        |
