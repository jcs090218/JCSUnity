# JCS_DropdownLanguage

A dropdown menu let you choose the language.

## Variables

| Name              | Description                                         |
|:------------------|:----------------------------------------------------|
| mOptions          | List of supported languages.                        |
| mRemoveAllOptions | If true, remove all other options at the beginning. |

## Functions

| Name    | Description               |
|:--------|:--------------------------|
| Refresh | Refresh all options once. |
