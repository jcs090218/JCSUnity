# JCS_DeactivePanelGamePadButton

Button that deactive the panel. (Gamepad)

## Variables

| Name             | Description                   |
|:-----------------|:------------------------------|
| mDialogueObjects | Panels to be deactive.        |
| mTweenPanels     | Tween Panels to be deactive." |
| PlaySound        | Play dialogue sound.          |
