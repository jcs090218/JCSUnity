# JCS_UpdateSliderButton

Update the slider value.

## Variables

| Name         | Description                            |
|:-------------|:---------------------------------------|
| mTargetValue | Target value to assign to slider list. |
| sliders      | List of effected sliders.              |

## Functions

| Name               | Description                     |
|:-------------------|:--------------------------------|
| UpdateSlidesrValue | Update all sliders' value once. |
