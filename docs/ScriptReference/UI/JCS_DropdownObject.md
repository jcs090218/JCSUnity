# JCS_DropdownObject

Holds all type of UI dropdown objects.

## Variables

| Name            | Description                     |
|:----------------|:--------------------------------|
| mDropdownLegacy | Target dropdown renderer.       |
| mTMP_Dropdown   | Target dropdown renderer. (TMP) |

## Functions

| Name         | Description            |
|:-------------|:-----------------------|
| value        | Return dropdown value. |
| ClearOptions | Clear all options.     |
